# Task Management API

A **Task Management API** built with **FastAPI** and **Python** that provides secure user authentication and complete task management functionality. The project implements **JWT token-based authentication**, allowing users to register, log in, and access protected endpoints using generated access tokens.

The API supports full **CRUD (Create, Read, Update, Delete)** operations for managing tasks, enabling authenticated users to create new tasks, view their tasks, update task details, and delete tasks. Built with **FastAPI**, **Pydantic**, and a relational database, the project follows a clean and modular architecture, making it easy to maintain and extend.

## Features

* 🚀 Built with **FastAPI** and **Python**
* 🔐 User Registration and Login
* 🎫 JWT Authentication with Access Token Generation
* 🔒 Protected API Endpoints using Bearer Token Authentication
* ✅ Complete Task CRUD Operations
* 📋 Create, Read, Update, and Delete Tasks
* ✔️ Request and Response Validation using Pydantic
* 🗄️ Database Integration with SQLAlchemy
* 📄 Interactive API Documentation with Swagger UI and ReDoc
* ⚡ High-performance RESTful API
* 🧩 Clean, modular, and scalable project structure
* 🛠️ Easy to customize and extend for larger applications

This project is ideal for learning **FastAPI**, **JWT authentication**, **REST API development**, **database integration**, and implementing secure backend services using modern Python best practices.
